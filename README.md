# 🚀 Cosmic Dodge

Cosmic Dodge is a browser-based space shooter game controlled using your **webcam hand gestures**, **MediaPipe hand tracking**, and a **Teachable Machine image model**.

Move your hand to control the spaceship, dodge meteors, collect power-ups, and survive as long as possible.

---

## 🎮 Features

- 🖐️ Webcam-based hand tracking using MediaPipe Hands
- 🤖 Gesture recognition using Teachable Machine
- ⌨️ Keyboard fallback controls
- 🚀 Smooth spaceship movement using index finger tracking
- ☄️ Random meteor spawning and meteor splitting
- 🔫 Laser shooting system
- 🛡️ Shield system
- ❤️ Hull integrity / health system
- ⚡ Power-ups:
  - Shield Charge
  - Double Shot
  - Repair
- 🔊 Retro space sound effects using Web Audio API
- 🏆 High score saved using Local Storage
- 📈 Difficulty increases with levels
- 🌌 Animated space background and particle effects
